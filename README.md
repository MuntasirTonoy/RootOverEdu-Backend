# RootNahid Backend

This is the Express.js / MongoDB backend for the RootNahid e-learning platform.

## Features
- RESTful API for Movies/Subjects/Courses
- Admin and User roles
- Firebase Authentication Middleware

## Deployment
- **Backend URL**: [https://root-nahid-backend.vercel.app](https://root-nahid-backend.vercel.app)
- **Frontend URL**: [https://roottonahid.vercel.app](https://roottonahid.vercel.app)

## Setup
1. `npm install`
2. Create `.env` with:
   - `MONGO_URI`
   - `PORT`
3. `npm run dev`
